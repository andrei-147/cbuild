## đźš€ The Concept

`cbuild` operates on a simple principle: **If it's in the folder and has a valid extension, it gets compiled.** It recursively crawls your project directory to identify source files and automatically includes parent directories for headers.

## âś¨ Features

* **Zero Config:** No `.json`, `.toml`, or `Makefile` required.
* **Auto-Inclusion:** Automatically detects header and library directories.
* **Recursive Discovery:** Finds files in nested subdirectories.
* **Exclusion Rules:** Automatically ignores hidden files/folders starting with `.` or `_`.
* **Flexibility:** Has many flags that can be used to customize to the user's preference.

## đź’» Usage

### Basic Build

```bash
# Basic debug build using g++
./cbuild -Bdebug

# Specify a different compiler (e.g., clang++)
./cbuild -Brelease -Cclang++

# Specify the C / C++ standard
./cbuild -Brelease -Cg++ -Sc++23
```

### Flags

1. **Mode:** [-B] `debug` or `release`. (e.g. `-Bdebug`)
2. **Compiler:** [-C] Define which compiler to invoke. Defaults to `g++`. (e.g. `-Cclang`)
3. **Standard:** [-S] Specify the C/C++ standard used when building the project. Defaults to `c++17`. (e.g. `-Sc++23`)
4. **Libraries:** [-L] Specify a library you want to link. (e.g. `-LSDL3`)
5. **Executable:** [-O] Specify the name of the executable produced. Defaults to `exec`. (e.g. `-Ocbuild`)
6. **Excludes:** [-X] Specify any file/directory to exclude from the build process. (e.g. `-Xexamples`)
7. **Help:** [--help] Specify to see this list explained briefly. (e.g. `--help`)

## đź› ď¸Ź How it works

1. **Cleanup:** Removes previous binary to ensure a fresh build.
2. **Scan:** Recursively walks the current path.
3. **Files:** Collects source files, include directories and library directories each into its respective set.
4. **Filter:** Ignores directories and files starting with `.` or `_`.

5. **Execute:** Constructs a system command and executes it:
`[compiler] [src_files...] -I[header_dirs...] -L[lib_dirs...] -l[linked_libs...] -o exec [debug flags... | release flags...] -std=[c/c++xx]`
